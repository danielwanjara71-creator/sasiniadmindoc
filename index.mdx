# Introduction

The **Sasini Admin Dashboard** is the web-based management centre for the Sasini PLC platform. It lets authorised staff manage farmer accounts, track produce pickups, oversee Agri-Shop orders, broadcast communications, monitor pest threats, and control all aspects of the farmer-facing mobile experience.

<Note>
This dashboard is **desktop-only** (minimum screen width 991px). Always use a laptop or desktop computer.
</Note>
